# 📊 YouTube Channel Analysis (Power BI)

## 📌 Project Overview
This project focuses on analyzing YouTube channel data using Power BI to understand performance, engagement, and growth trends. The dashboard provides insights into video performance, subscriber base, and revenue estimation.

## 🔧 Tools Used
- Power BI
- Excel / CSV Dataset

## 📊 Key Insights
- Analysis of total number of videos and their impact on performance
- Ranking analysis to identify top-performing channels/videos
- Subscriber growth and distribution patterns
- Estimated monthly earnings insights
- Country-wise performance comparison

## 📈 Dashboard Features
- Interactive visuals for better understanding
- Filters for country and channel analysis
- Clear representation of growth trends
- KPI indicators for quick insights
